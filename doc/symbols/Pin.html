{Number} getNumber()
Get the pin number corresponding to the Arduino documentation for the type of board. The number has been adjusted to match the Arduino documentation rather than truly represent the pin number of the AVR. For example, with a Standard Arduino board, analog pin 0 = digital pin 14 even though analog pin 0 is actually pin 16 (because there are 3 ports and it's the first pin of the 3rd port). This will be different for other board types such as the Arduino Mega.
Returns:
{Number} The pin number
